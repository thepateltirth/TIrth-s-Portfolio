SpeedyBites

Full-Stack

A full-stack food ordering web application that lets users browse restaurants, add items to a real-time cart, and place orders seamlessly. Built with a React frontend consuming a Node.js + Express REST API, backed by MongoDB for persistent data storage.

Key Features
  • ✦  JWT-based user authentication & session management
  • ✦  Real-time cart with quantity controls and order summary
  • ✦  RESTful API with full CRUD for menus and orders
  • ✦  MongoDB + Mongoose for structured document storage
  • ✦  Mobile-first responsive UI

Facial Emotion Recognition

ML / AI

A deep learning system that classifies human emotions from facial images using a Convolutional Neural Network. Trained on the FER-2013 dataset, the model distinguishes 7 emotional states in real-time — suitable for accessibility tools, interactive apps, and research.

Key Features
  • ✦  CNN with Conv2D + MaxPooling + Dropout layers
  • ✦  Classifies 7 emotions: angry, disgust, fear, happy, sad, surprise, neutral
  • ✦  Data augmentation for better generalization
  • ✦  Real-time inference via webcam using OpenCV
  • ✦  Trained on FER-2013 (35k+ images)
