Hey, I'm Tirth 👋

I'm a full-stack developer who loves building things that live on the internet — whether that's a seamless food ordering app or a deep learning model that reads human emotion. I enjoy the full journey from wireframe to deployed product.

My journey started with curiosity about how websites work, which quickly expanded into backend systems, databases, and eventually machine learning. I'm comfortable on both sides of the stack and genuinely enjoy switching contexts between a React component and a Python model.

I believe great software is clear, fast, and honest — it does what it says, works where it needs to, and doesn't make users think too hard. That philosophy shapes every project I take on.
